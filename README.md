# Roleta V4

Roleta de premios da V4 Alfradique em React + Vite.

## O que tem

- Lista manual de premios, um por linha.
- Fatias ajustadas automaticamente pela quantidade de premios.
- Logo V4 no centro da roleta.
- Premio sorteado anunciado na tela.
- Persistencia local dos premios no navegador.
- Fundo escuro com grade vermelha no estilo V4.

## Como rodar

```bash
npm install
npm run dev
```

## Cloudflare Pages

- Build command: `npm run build`
- Build output directory: `dist`
- Root directory: vazio
- Branch: `main`

Se o build pedir uma versao de Node, use a variavel:

```text
NODE_VERSION=24.11.1
```
